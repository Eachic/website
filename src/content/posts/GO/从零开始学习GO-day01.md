# 从零开始学习GO day01

今天完成Go基础语法的学习，然后记录一下一些需要记忆的GO的语法特点
## 基础数据类型
- int系列（uint,uint32,uint64 等）
- float系列（float32,float64 等）
- string （引用对象，16字节，8字节真实内存指针，8字节长度）
- 数组 （值对象，大小不可变）
- slice （引用对象，类似vector，大小可变）
- map (引用对象，和c++ map 类似)
- struct (自定义类型，)
- interface (接口， 规定了一系列函数，)
- method 与 function



## 语法特点
- go中局部变量声明必须使用，否则会报错
- go的函数不支持重载
